# LOG.md

## 2026-04-22 — Project start

### Name decision

Project name defined as **Alvarium**.

Reason:

* inspired by the name Álvaro
* represents protection and control
* strong branding potential

---

### Product direction

Alvarium will NOT be a cPanel clone.

Instead:

* modern hosting platform
* lightweight
* focused on real-world usage

---

### Tech stack chosen

* Next.js (frontend)
* NestJS (backend)
* PostgreSQL
* Redis
* Go (agent)

---

### Architecture

Defined as:

* Core (API + Web)
* Agent (runs on nodes)

---

### License

AGPLv3 chosen to:

* allow community contributions
* prevent closed SaaS forks

---

### UI Direction

* modern SaaS style
* light + dark mode
* minimalistic dashboard
* focus on operational data

---

### Security strategy

Security must include:

* JWT authentication
* RBAC
* audit logs
* secure agent communication

---

### Initial focus

Start with:

* project structure
* dashboard UI (mock)
* auth system
* basic entities (users, plans, accounts)

---

### Future vision

* open core + SaaS model
* automation and intelligence
* strong observability
* operational alert system
